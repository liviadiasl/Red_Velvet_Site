# Red_Velvet_Site

## Sobre o projeto
Projeto desenvolvido com o objetivo de praticar fundamentos de desenvolvimento web, com foco na estruturação de páginas, organização de conteúdo e estilização utilizando HTML e CSS.

O site apresenta informações sobre o grupo Red Velvet, permitindo a navegação entre páginas por meio do cabeçalho e rodapé, além do acesso a links externos e redes sociais.

---

## Funcionalidades
- Navegação entre páginas através de menu e rodapé
- Acesso a links externos e redes sociais
- Exibição organizada de textos e imagens
- Interação básica com botões e campos de texto

---

## Tecnologias utilizadas
- HTML5
- CSS3

---

## Como executar o projeto
1. Clone este repositório:
```bash
git clone https://github.com/liviadiasl/Red_Velvet_Site.git
```
2. Abra o arquivo index.html no navegador.
